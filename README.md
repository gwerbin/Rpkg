# R CLI package manager

## Installation

1. Install R (version >3.4 highly recommended)
2. `ln -s ./Rpkg.R ~/bin` or something along those lines

## Instructions

```
Commands:
    help
    install / add
    upgrade / update
    uninstall / remove
    search (not implemented)

Options:
    -V / --version
    -h / --help
```
